# 🚀 Тестовое задание в Autodoc.ru

> Элегантное и производительное iOS-приложение для отображения новостной ленты 📱✨

---

## 🧩 Описание проекта

Необходимо разработать приложение, отображающее **новостную ленту** с современным UI и плавным UX.

📡 **Источник данных:**
https://webapi.autodoc.ru/api/news/1/15


---

## 🛠 Используемые технологии

Проект реализован с использованием актуального стека iOS-разработки:

- 🧠 **MVVM + Combine**
- 📦 **UICollectionView**
- 🧩 **CompositionalLayout**
- ⚡ **DiffableDataSource**
- ⏳ **async/await**

---

## 🎨 Внешний вид

Приложение следует современным гайдлайнам Apple 🍏

### 📰 Ячейка новости:
- 🖼 Изображение
- 📝 Заголовок

### 📖 Поведение:
- При нажатии открывается **полный текст новости**


## ⚙️ Требования

✔️ Постраничная загрузка данных (Pagination)  
✔️ Все сетевые запросы через **async/await**  
✔️ 📱 Оптимизация под iPad  
✔️ 🚫 Без сторонних библиотек  
*(Alamofire, Kingfisher и др. не используются)*

## 💡 Особенности реализации

- Плавная и отзывчивая прокрутка списка
- Эффективное обновление UI через DiffableDataSource
- Асинхронная загрузка изображений без блокировки UI
- Чёткое разделение слоёв (MVVM)
